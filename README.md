# Scrapy-Crawlers
Web Scraping using Python, Scrapy and Splash

## Installation
- Cloning Repo:
```
git clone https://github.com/ADI-KOTKAR/Scrapy-Crawlers.git
```
- Installing Requirements:
 ```
pip install requirements.txt
```
